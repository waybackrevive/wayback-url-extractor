# Example: Extract URLs for SEO Analysis

## Common SEO Use Cases

### 1. Find All HTML Pages
```bash
python extractor.py example.com --filter "*.html" --format csv
```

### 2. Historical Content Audit (Last 5 Years)
```bash
python extractor.py example.com --from 2019 --to 2024 --status 200 --format csv
```

### 3. Find Lost Blog Posts
```bash
python extractor.py example.com --filter "*blog*" --format csv
```

### 4. Complete Site Inventory
```bash
python extractor.py example.com --limit 50000 --format json
```

## Analysis Tips

### In Excel/Google Sheets
1. Open the CSV file
2. Use filters to find specific URL patterns
3. Sort by timestamp to see content evolution
4. Create pivot tables for content categorization

### For SEO Strategy
- Compare with current sitemap
- Identify 404s that should be recovered
- Find old URLs for 301 redirects
- Discover forgotten valuable content
